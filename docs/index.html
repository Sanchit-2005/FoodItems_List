🍔 Food Items List

Welcome to the Food Items List application!

Note:

This is a static preview hosted on GitHub Pages. The full application with backend functionality requires a Node.js server.

To run locally:

  1. Clone the repository
  2. Run: npm install
  3. Run: node app.js
  4. Visit: http://localhost:8080

Features:

  • View food items list
  • Add new food items with photos
  • Mark items as available/unavailable
